# Security Policy

## Scope / Alcance

Este repositorio (`bolivar-stablecoin-research`) es una colección de documentos, diagramas y contratos inteligentes de prueba. **No es un sistema en producción.**

Riesgos considerados relevantes para reportar:

- Exposición accidental de credenciales reales.
- Información personal (PII) en los datos de ejemplo.
- Enlaces maliciosos en la documentación.

Riesgos fuera de alcance:

- Vulnerabilidades teóricas en los snippets de Solidity (son ilustrativos).
- Ataques a las redes de prueba (testnets) mencionadas.

## Reporting a Vulnerability / Reportar Vulnerabilidad

Si encuentras información sensible expuesta, por favor **NO abras un Issue público**.

Envía un correo inmediatamente a: `security@antigravity.dev` (o contacta al maintainer principal).

Intentaremos responder en menos de 48 horas.
